# AI Career Advisor using n8n and Google Gemini

## Overview

AI Career Advisor is an intelligent career guidance assistant built using n8n, Google Gemini, and Google Sheets. The system helps students and professionals discover suitable career paths, identify required skills, explore salary expectations, and receive personalized learning roadmaps based on their interests and goals.

## Features

* AI-powered career guidance
* Personalized career recommendations
* Skill gap analysis
* Learning roadmap generation
* Salary insights for different career paths
* Project and certification recommendations
* Integration with Google Sheets as a dynamic knowledge base
* Conversational chat interface using n8n AI Agent

## Tech Stack

* n8n
* Google Gemini
* Google Sheets
* AI Agent Framework
* Workflow Automation

## Workflow

1. User submits a career-related query through the chat interface.
2. AI Agent processes the request.
3. Google Gemini generates intelligent responses.
4. Google Sheets provides career, skills, salary, and roadmap data.
5. The system returns personalized career recommendations and action plans.

## Use Cases
